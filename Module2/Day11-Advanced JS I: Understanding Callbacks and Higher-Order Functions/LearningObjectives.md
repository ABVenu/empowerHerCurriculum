# **Advanced JS I: Understanding Callbacks and Higher-Order Functions**

#### Learning Objectives

## **1. Callbacks**

- Understand what a callback function is
- Identify why callbacks are needed
- Explain how callbacks help in reusing logic
- Write simple callback examples (e.g., passing `greet` into `print`)

---

## **2. Higher-Order Functions (HOFs)**

- Understand what a higher-order function is
- Identify why HOFs improve abstraction and code cleanliness
- Recognize where HOFs are commonly used in JavaScript

---

## **3. Array Higher-Order Functions**

- **map**

  - Transform values and return a new array

- **forEach**

  - Iterate over items for side effects

- **filter**

  - Keep items that satisfy a condition

- **reduce**

  - Combine values into a single result

- **sort**

  - Sort elements using a compare callback
