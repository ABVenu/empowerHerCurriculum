# **Advanced JS II: How JavaScript Works**

#### Learning Objectives

### **1. Scoping**

* Understand global, function, and block scope
* Identify how scope determines variable visibility
* Recognize common scoping pitfalls

---

### **2. Hoisting**

* Understand hoisting behavior for `var`, `let`, `const`, and functions
* Identify the Temporal Dead Zone (TDZ)
* Predict code execution outcomes using hoisting rules

---

### **3. Lexical Environment**

* Understand what a lexical environment is
* Identify how variables are stored in lexical scopes
* Recognize how lexical environments chain together

---

### **4. Closures**

* Understand what a closure is
* Identify how inner functions retain access to outer lexical environments
* Recognize practical uses of closures

---

### **5. Execution Context**

* Understand what an execution context is
* Identify the memory creation phase
* Identify the execution phase
* Understand how stack frames are created and destroyed

---

### **6. Memory Management**

* Differentiate between stack memory and heap memory
* Understand how primitive and reference values are stored
* Recognize how garbage collection works in JavaScript
