# **Modern JavaScript ES6+ Features**

#### Learning Objectives

By the end of Day 1, students will be able to confidently use the following ES6+ features:

---

## **1. Brief History of ECMAScript (ES)**

- Understand what ECMAScript (ES) means
- Identify why ES exists and what it standardizes
- Recognize major ES versions and evolution

---

## **2. What ES6 (ECMAScript 2015) Introduced**

- Identify key ES6 feature categories
- Understand why ES6 is a major update
- Recognize how ES6 improved JavaScript development

---

## **3. let, const & Block Scope**

- Differentiate between `var`, `let`, and `const`
- Understand Temporal Dead Zone (TDZ)
- Apply best-practice usage

---

## **4. Template Literals**

- Use multi-line strings
- Use string interpolation
- Understand tagged template basics

---

## **5. Arrow Functions**

- Write arrow functions with different syntaxes
- Use implicit return
- Understand `this` behavior changes
- Identify when not to use arrow functions

---

## **6. Ternary Operator**

- Apply basic ternary syntax
- Avoid nested ternaries
- Use ternary in practical conditions

---

## **7. Destructuring**

- Apply array destructuring
- Apply object destructuring
- Use default values
- Use nested destructuring
- Use function parameter destructuring

---

## **8. Spread & Rest Operators**

- Use spread in arrays, objects, and functions
- Use rest in function parameters
- Apply common patterns: cloning, merging, removing keys

---

## **9. Enhanced Object Literals**

- Use property shorthand
- Use computed property names
- Define methods concisely

---

## **10. Default Parameters**

- Use default function parameter values
- Avoid undefined vs falsy confusion

---

## **11. Optional Chaining**

- Use `?.` for safe property access
- Apply optional chaining in nested objects and API responses

---
