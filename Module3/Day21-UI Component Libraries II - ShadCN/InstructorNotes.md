# **Learning Objectives**

---

## **Understand the Role of External UI Component Libraries**

- Explain why external UI component libraries are used in modern applications.
- Understand how component libraries improve:

  - Development speed
  - UI consistency
  - Accessibility

- Compare building components from scratch vs using pre-built components.

---

## **Understand What shadcn/ui Is**

- Explain what shadcn/ui is and how it differs from traditional UI libraries.
- Understand shadcn/ui as:

  - A collection of reusable components
  - Built on top of Tailwind CSS

- Identify why shadcn/ui components are copied into the project instead of being installed as a dependency.

---

## **Set Up and Use shadcn/ui in a React Project**

- Initialize shadcn/ui in a React project.
- Understand the folder structure created for components.
- Integrate shadcn/ui with Tailwind CSS configuration.

---

## **Use Core shadcn/ui Components**

Learners should gain hands-on familiarity with commonly used components such as:

- Button
- Input
- Label
- Card
- Badge
- Avatar
- Separator

Understand:

- Default styling and variants
- Props used to customize behavior and appearance

---

## **Work with Layout and Utility Components**

- Use layout-related components such as:

  - Card layouts
  - Containers
  - Sections

- Combine shadcn/ui components with Tailwind utility classes.
- Build structured UI blocks using composable components.

---

## **Use Interactive and Feedback Components**

- Use components that handle user interaction, such as:

  - Dialog / Modal
  - Dropdown Menu
  - Tabs
  - Tooltip

- Understand how these components improve user experience and accessibility.

---

## **Build Forms Using shadcn/ui Components**

- Use form-related components to build consistent forms.
- Understand how inputs, labels, and buttons work together.
- Apply controlled component concepts with shadcn/ui inputs.

---

## **Understand Accessibility and Best Practices**

- Understand the importance of accessibility in UI components.
- Learn how shadcn/ui provides accessible components by default.
- Follow best practices when integrating and customizing components.

---

## **Understand Customization and Extension**

- Modify shadcn/ui components safely.
- Extend components using Tailwind utility classes.
- Maintain consistency while customizing UI components.

---

## **End-of-Session Expected Outcomes**

By the end of this session, learners will be able to:

- Explain the importance of using external UI component libraries
- Set up and use shadcn/ui in a React project
- Use and customize core UI components confidently
- Combine shadcn/ui components with Tailwind CSS
- Build consistent and accessible user interfaces quickly
