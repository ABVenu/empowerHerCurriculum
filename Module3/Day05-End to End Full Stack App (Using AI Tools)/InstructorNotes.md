### Lecture Name

## End-to-End Full Stack App (Using AI Tools)

## Learning Objectives

1. **Plan and build a complete Full Stack Web Application** with multiple features such as recipe creation, listing, and rating.
2. **Use AI tools effectively** (e.g., ChatGPT, GitHub Copilot, FireCMS UI, design assistants) to accelerate development and productivity.
3. Apply modular and reusable **frontend architecture** with components like navigation bar, dashboard sections, and recipe cards.
4. Configure and integrate a **backend database** (Firebase) to support:

   - Create, Read, Update, Delete (CRUD) operations for recipes
   - Public/Private visibility settings

5. Implement **Shareable URLs** for viewing recipe details publicly without authentication.
6. Enable users to **rate and review recipes**, and display aggregated ratings in the UI.
7. Build a **Dashboard screen** for authenticated users to manage their own recipe content.
8. Maintain **clean project organization** using:

   - Dedicated modules for API calls
   - Constants in a metadata/config file
   - Separate folders for pages, scripts, assets, and styles

9. **Version-control** the entire application using Git and GitHub.
10. **Deploy** the live application using GitHub Pages or Firebase Hosting for public access.
11. Write a professional **README documentation** with app description, features, installation steps, and live demo link.
